Frequently Asked Questions

FAQs:

How do I start my first Hunger Games?🔗

The easiest way to start a game is by using the ?hg autoplay command.
This will start a game with default settings and automatically continue until the game is over.

How do I create teams in HG?🔗

To enable teams, set the `teamSize` option to the size of teams you wish to have.
?hg opt teamSize 3 will automatically form teams with 3 players maximum.
After changing the team size, you can reset the teams to automatically create the teams using ?hg reset teams. If you have manually created teams previously, you may not want to do this.

Once teams are created you can manage the players on the teams using the HG webview or with the ?hg teams commands.

How do I turn off teams in HG?🔗

To disable teams, set the `teamSize` option to 0.
?hg opt teamSize 0

Is there a list of HG weapons somewhere?🔗

The HG webview page has all default and custom events for your server under the Events tab.
You can also customize your events and weapons here as well.

Weapons by default may include: poison, throwing knives, rope, sword, toothbrush, hatchet, mace, wood, shovel, machete, knife, spear, explosive, grenade, gun, bow, slingshot, and alcohol.

What if I don't want someone to use a command?🔗

All permissions for commands are managed using the enable, disable, and show commands.
To prevent someone from using a command, they cannot have any of the permissions in the "allowed with" section, or they can have any blocking permissions in the "blocked for" section.

If any permission is matched in the "allowed with" section, they will have access to the command.
If no permission is matched in the "allowed with" section, they will not have access to the command.
If any permission is matched in the "blocked for" section, they will not have access to the command.
If no permission is matched in the "blocked for" section, they will have access to the command.

To change "allowed with" to "blocked for" or vice versa, use ?disable {COMMAND} everyone or ?enable {COMMAND} everyone respectively.

How do I allow other users to use Hunger Games commands?🔗

All permissions for commands are managed using the enable, disable, and show commands.
Most cases you can just type ?enable hg * * @SpikeyRobot#0001, where SpikeyRobot is the user to let manage the Games.

These can be confusing to use. If you still need help after reading the examples, feel free to join the Discord server to ask for help (or use the pmspikey command).

Which command do I use to solve my math problem?🔗

?eval will probably be the most common one to use. It does math, and converts units, but does not work with variables/algebra.
?solve can solve for multiple variables in an equation and does algebra.

Why does SpikeyBot reply with a random number after some messages?🔗

SpikeyBot by default will count the number of times the word "rigged" has been said.
This can be disabled using the ?togglerigged command.

This started as a joke due to people getting mad when they didn't win in HG.
